Concept

AI 사이에 진짜 적이 섞여 있다

맵 위에 40명의 AI가 자동 스폰.
그 중 일부는 처치 대상, 일부는 건드리면 안 되는 인물.
제한 시간 120초. 한 자루 권총. 점수가 곧 실력.

핵심 룰

· 학생 / 마스코트 → 사격 = 가산점
· 교수님 → 사격 = 감점
· 시간 종료 = 게임 오버
· RESTART 타겟 사격 시 즉시 재시작

Spawn Logic

맵 전체에 40명 자동 배치

NavMesh 면적 비례 샘플링

베이크된 walkable 영역의 삼각형 면적에 비례해 랜덤 위치 선택. 맵 모양/크기 무관 균등 분포.

가중치 + 최대치

각 variant 의 weight 비례 추첨. maxCount 도달한 종류는 후보에서 제외.

자동 리스폰

3초마다 생존 수 체크 → 부족분만 새 위치에 1마리씩 spawn. 항상 40명 유지.

사망 이펙트

처치 시 AI 로고 스프라이트(claude · gemini 등)가 사방으로 튀어 오름. variant 별 모션 override 지원.

Architecture

시스템 구성

StartTarget
권총 raycast hit 1회 → GameSession.StartGame() 호출. 첫 진입 시에만 표시.
GameSession
120초 카운트다운 · BGM loop · 처치 수 집계 · OnGameStarted / OnTimeUp 이벤트 발신.
EnemySpawner
NavMesh 면적 비례 spawn · weight/maxCount 기반 variant 선택 · 정적 점수 시스템 통합.
EnemyHealth
개별 AI 체력 · 사망 처리 · variant 별 score / death motion override 수신.
PistolMecanic
XR Toolkit grab · 트리거 발사 · 머즐 플래시 · 탄피 · 히트 이펙트.
ScoreBoardUI · ScorePopup
World Space UI · 처치 위치에 점수 popup · 누적 점수 실시간 표시.
GameOverUI · RestartTarget
시간 종료 시 결과창 · RESTART 버튼 또는 재시작 큐브 사격 시 다음 라운드.
