3 카메라 · 동기화 · 편집

  • 카메라 — 삼각대 고정, 와이드. 배우 동선(침대→걷기)이 프레임 안 벗어나게 리허설.
  • 화면녹화 — QuickTime "새로운 화면 기록", 대시보드 창. "액션"과 동시 시작.
  • 싱크 — 시작 때 손뼉 1번 → 영상·화면녹화 소리로 정렬. 센서 로그 타임스탬프도 참고.
  • 편집 — 좌(카메라)/우(대시보드) 나란히, 손뼉 지점 맞춰 정렬. ~40초로 트림.
  • 챕터 — 덱 슬라이드 하단 챕터바가 ①②③ 표시 대신함(영상 자막 최소).
  • 여러 테이크 — NG 대비 3~4회. 제일 깔끔한 것 채택.
필수 선결 — 촬영 전 USB 안정 확인(5분 연속 데이터 흐르는지) · 리허설에서 불안정 보행이 지표에 실제로 잡히는지(안 잡히면 임계 조정).
