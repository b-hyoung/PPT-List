04 · 핵심 차별점

위치(ToF) + 자세(mmWave)로 이탈 전조를 단계로 나눈다

05 · 근거 (왜 야간 침상인가)

야간 낙상의 상당수는 "밤에 혼자 일어서다"

치매 낙상 1.7~3배

대조군 비교 — 치매 어르신은 비치매 대비 낙상 1.7~3배 (van Doorn 2003 / BMC Geriatrics 2021)

야간 수면장애·배회

치매 어르신의 야간 뒤척임·배회가 압도적으로 많음 (Webster 2019 · Guu 2025)

화장실 가려다 낙상

노인 야간낙상의 다수 = 밤에 화장실 가려 혼자 일어서는 순간 (Rose 2020)

공백 · "치매가 더 넘어지는 이유 = 야간 침대이탈"을 정량 입증한 연구는 아직 없다 → 마주봄이 그 데이터를 모은다.
06 · 시스템 — 역할 분리

3센서가 각자 다른 일을 하고, 서버가 합친다

ToF · 위치

침상 거리격자로 몸이 침대 어디에·가장자리로 쏠렸나. 이탈 게이트.

mmWave · 자세

점군으로 눕/앉/일어섬 자세 + 낙상(높이 급강하). 이탈 더블체크.

CSI · 개인 위험도

자는 동안 안정시 HR·호흡 추세 → 그날 개인 위험도를 가중(감지 아님, 노브).

3센서 측정
임베디드
서버 융합
baseline·z-score·위험도 가중
간호사 대시보드
4단계 알림
슬로건 · 임베디드 = 측정, 서버 = 판정·표출. mmWave=이탈/자세 · ToF=위치 · CSI=개인화 가중.
07 · 시연 ① — 베이스라인

정상 취침 — 센서 3종이 평상시 뭘 보나

▶ 시연 영상 자리 · S1
▶
[촬영] 어르신이 침대에 누워 안정. ToF=재실 / mmWave=조용 / CSI가 HR·호흡 기준선을 잡는 화면. → 이 사람의 개인 baseline 확보.
✓ CSI 현황 — 단일 안테나 ESP32로 심박 측정을 시도했으나 다중안테나 기반 작업이 필요한 기기의 한계를 확인. 현재는 호흡·재실 신호 중심, 다중안테나 PoC는 향후 과제.
07 · 시연 ② — 이탈 감지 → 알림 (핵심)

뒤척임은 넘기고, 진짜 전조에서 알린다

▶ 시연 영상 자리 · S2
▶
[촬영] 누움→뒤척임(알림 X)→가장자리 쏠림(ToF)→상체 일어남(mmWave 더블체크)→간호사 대시보드 알림. 단계가 색으로 올라가는 화면.
✓ 우리 실측·검증 — 침대 안/밖(이탈) 구분 겹침 0으로 분리(우리 데이터) · 뒤척임은 위치 유지로 오탐 제외 · 통제 라벨 프로토콜로 수집.
07 · 시연 ③ — 위험도 개인화

같은 행동, 고위험 환자엔 더 빠르게

▶ 시연 영상 자리 · S3
▶
[촬영] 두 환자가 같은 이탈 행동. 그날 CSI 생체가 나쁜 고위험 환자는 더 이른 단계에서·더 높은 우선순위로 알림이 뜨는 대비 화면.
✓ 차별점 실연 — 상용은 대개 이벤트 감지에 그침. 우리는 CSI 개인 baseline으로 알림 민감도를 개인·그날에 맞춰 조정.

09 · 정직한 한계

못 하는 것을 숨기지 않는다

CSI 기기의 한계

단일 안테나 ESP32로 심박 측정을 시도했으나, 정밀 측정은 다중안테나 기반(두 안테나 CSI 비율로 노이즈 상쇄)이 필요함을 확인 — 학술적으로도 동일(IEEE BHI 2021 등). → 다중안테나 PoC는 향후 과제.

자세 미세구분

센서 장착 각도에 민감. 눕/앉 미세구분은 데이터 더 필요. 침대 안/밖은 견고.

데이터 축적

현장 반복 데이터로 정확도↑. "융합이 오탐 줄이나"는 실증으로 답할 것.

✓ 한계를 명시하는 것 자체가 신뢰 — 검증 조건(통제/현장)을 분리해 과대주장을 피한다.
10 · 로드맵 & 마무리

데이터로 "융합이 낫다"를 증명하는 방향

  • 현장 반복 데이터 축적 → 뒤척임 vs 이탈전조 라벨셋 구축
  • 융합 실증 — ToF+mmWave가 레이더 단독 대비 오탐 줄이나 수치로
  • 시계열 모델(mPCT-LSTM/PointNet++) — 전이 정밀화 · CSI 개인 위험도 가중 통합
  • 무선화(MQTT) · 다중안테나 CSI PoC · 대시보드 고도화
